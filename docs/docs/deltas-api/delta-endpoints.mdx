## Get Global Delta Leaderboards

<Endpoint verb="GET" path="/deltas/leaderboard" />
<br />

Fetches the current top deltas leaderboard for a specific metric and period. Returns an array of [DeltaLeaderboardEntry](/deltas-api/delta-type-definitions#object-delta-leaderboard-entry) objects.

<br />

**Query Params**

| Field       | Type                                                                   | Required | Description                                  |
| ----------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------- |
| period      | [Period](/global-type-definitions#enum-period)                         | `true`   | The delta's time period.                     |
| metric      | [Metric](/global-type-definitions#enum-metric)                         | `true`   | The delta's metric.                          |
| playerType  | [PlayerType](/players-api/player-type-definitions#enum-player-type)?   | `false`  | The player's account type to filter by.      |
| playerBuild | [PlayerBuild](/players-api/player-type-definitions#enum-player-build)? | `false`  | The player's account build to filter by.     |
| country     | [Country](/players-api/player-type-definitions#enum-country)?          | `false`  | The player's country of origin to filter by. |

<br />

**Example Request**

<TabbedCodeBlock>

```curl
curl -X GET https://api.wiseoldman.net/v2/deltas/leaderboard?metric=agility&period=week&playerType=hardcore \
  -H "Content-Type: application/json"
```

```javascript
const { WOMClient, Metric, Period, PlayerType } = require('@wise-old-man/utils');

const client = new WOMClient();

const deltasLeaderboard = await client.deltas.getDeltaLeaderboard({
  period: Period.WEEK,
  metric: Metric.AGILITY,
  playerType: PlayerType.HARDCORE
});
```

</TabbedCodeBlock>
