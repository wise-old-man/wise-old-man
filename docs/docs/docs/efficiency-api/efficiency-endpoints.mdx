## Get Global Efficiency Leaderboards

<Endpoint verb="GET" path="/efficiency/leaderboard" />
<br />

Fetches the current efficiency leaderboard for a specific efficiency metric. Returns an array of [Player](/players-api/player-type-definitions#object-player) objects.

<br />

**Query Params**

| Field       | Type                                                                         | Required | Description                                           |
| ----------- | ---------------------------------------------------------------------------- | -------- | ----------------------------------------------------- |
| metric      | [ComputedMetric](/global-type-definitions#enum-computed-metric) or `ehp+ehb` | `true`   | The efficiency metric.                                |
| playerType  | [PlayerType](/players-api/player-type-definitions#enum-player-type)?         | `false`  | The player's account type to filter by.               |
| playerBuild | [PlayerBuild](/players-api/player-type-definitions#enum-player-build)?       | `false`  | The player's account build to filter by.              |
| country     | [Country](/players-api/player-type-definitions#enum-country)?                | `false`  | The player's country of origin to filter by.          |
| limit       | integer                                                                      | `false`  | The pagination limit. See [Pagination](/#pagination)  |
| offset      | integer                                                                      | `false`  | The pagination offset. See [Pagination](/#pagination) |

<br />

**Example Request**

<TabbedCodeBlock>

```curl
curl -X GET https://api.wiseoldman.net/v2/efficiency/leaderboard?metric=ehp&playerType=hardcore \
  -H "Content-Type: application/json"
```

```javascript
const { WOMClient, Metric, PlayerType } = require('@wise-old-man/utils');

const client = new WOMClient();

const efficiencyLeaderboards = await client.efficiency.getEfficiencyLeaderboards({
  metric: Metric.EHP,
  playerType: PlayerType.HARDCORE
});
```

</TabbedCodeBlock>

## Get Efficiency Rates

<Endpoint verb="GET" path="/efficiency/rates" />
<br />

Fetches the current efficiency rates for skills and bosses. Returns an array of [SkillMetaConfig](/efficiency-api/efficiency-type-definitions#object-skill-meta-config) or [BossMetaConfig](/efficiency-api/efficiency-type-definitions#object-boss-meta-config) objects based on choice of metric.

<br />

**Query Params**

| Field  | Type                                                                                                  | Required | Description                             |
| ------ | ----------------------------------------------------------------------------------------------------- | -------- | --------------------------------------- |
| type   | [EfficiencyAlgorithmType](/efficiency-api/efficiency-type-definitions#enum-efficiency-algorithm-type) | `true`   | The efficiency type to fetch metas for. |
| metric | `ehp` or `ehb`                                                                                        | `true`   | The efficiency metric.                  |

<br />

**Example Request**

:::info
When using the JavaScript / TypeScript package, you don't need to specify the metric. They are two separate functions, `getEHBRates` and `getEHPRates`.
:::

<TabbedCodeBlock>

```curl
curl -X GET https://api.wiseoldman.net/v2/efficiency/rates?type=main&metric=ehb \
  -H "Content-Type: application/json"
```

```javascript
const { WOMClient, Metric, PlayerType } = require('@wise-old-man/utils');

const client = new WOMClient();

const ehbRates = await client.efficiency.getEHBRates({
  playerType: PlayerType.MAIN
});
```

</TabbedCodeBlock>
